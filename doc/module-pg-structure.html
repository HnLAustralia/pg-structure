Module: pg-structure

pg-structure

require("pg-structure")(host, database, username, password, options, callback)

Exports a PostgreSQL schema as a detailed object. Please see @see table object for callback argument that you will get as a result.

Parameters:
Name Type Description
host String

Hostname of the database.
database String

Database name
username String

Username for connecting to db.
password String

Password to connecting to db.
options Object

Options

Properties
Name Type Argument Default Description
schema String <optional>
 public

Schema of the database.
port number <optional>
 5432

Connection port of the database server.
callback exportCallback

Callback to handle database structure object.

Example
var pg-structure = require('pg-structure');
var util         = require('util');
pg-structure('localhost', 'db', 'user', 'password', { schema: 'public', port: 5432 }, function(result) {console.log(util.inspect(result, {depth: null}));} );
pg-structure('localhost', 'db', 'user', 'password', 'public', function(result) {console.log(JSON.stringify(result));} );
