pgStructure

pgStructure~getDB(pgOptions, [schemas]) ⇒ Promise.<T>

Kind: inner method of pgStructure
Returns: Promise.<T> - - Promise with signature (DB).

Param Type Default Description
pgOptions Object | pg.client node-postgres client or connection parameters. Parameters passed directly to node-postgres. See it for details.
pgOptions.database string Database name
[pgOptions.host] string "localhost" Hostname of the database.
[pgOptions.port] number 5432 Port of the database.
[pgOptions.user] string Username for connecting to db.
[pgOptions.password] string Password to connecting to db.
[pgOptions.ssl] boolean | Object false Pass the same options as tls.connect().
[schemas] Array.<string> [public] PostgreSQL schemas to be parsed.
